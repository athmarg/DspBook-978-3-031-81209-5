# Continuous and Discrete-Time Signals and Systems

This repository contains **Python / Jupyter notebooks** for the book:

**A. Margaris – Continuous and Discrete-Time Signals and Systems**  
ISBN 978-3-031-81209-5, Springer  
[Book page on Springer](https://link.springer.com/book/10.1007/978-3-031-81209-5)

---

## How to run the notebooks online

You can launch all interactive notebooks **without installing anything** using Binder:

[Open in Binder](https://mybinder.org/v2/gh/athmarg/DspBook-978-3-031-81209-5/main?labpath=index.ipynb)

> Clicking the link will open JupyterLab with `index.ipynb` loaded automatically.

---

## Repository structure

The notebooks are organized into **14 directories**, corresponding to the **13 chapters and the Appendix**:

- `Chapter01/` – Continuous-Time Signals (Pages 1-66)  
- `Chapter02/` – Continuous-Time Systems (Pages 67-184)  
- `Chapter03/` – Fourier Analysis in the Continuous-Time Domain (Pages 185-302)  
- `Chapter04/` – The Laplace Transform (Pages 303-373)  
- `Chapter05/` – System Analysis in the Frequency Domain (Pages 375-495)  
- `Chapter06/` – Discrete-Time Signals (Pages 497-562)  
- `Chapter07/` – Discrete-Time Systems (Pages 563-643)  
- `Chapter08/` – The Z Transform (Pages 645-692)  
- `Chapter09/` – Systems in the Z Domain: Digital Processing Techniques (Pages 693-781)  
- `Chapter10/` – Fourier Analysis in the Discrete-Time Domain (Pages 783-936)  
- `Chapter11/` – Design of Analog Filters (Pages 937-1115)  
- `Chapter12/` – Design of Digital Filters (Pages 1117-1388)  
- `Chapter13/` – Elements of Statistical Signal Processing (Pages 1389-1462)  
- `Appendix/` – Fundamental Mathematical Concepts

- `index.ipynb` – landing page with links to all exercises and chapters

---

## Notes

- All notebooks are intended to be run in **JupyterLab / Jupyter Notebook**.  
- Users do **not need to install Python locally**; everything runs via Binder.
